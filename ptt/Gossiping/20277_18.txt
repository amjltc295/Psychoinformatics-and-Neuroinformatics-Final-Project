    <div id="main-content" class="bbs-screen bbs-content"><div class="article-metaline"><span class="article-meta-tag">作者</span><span class="article-meta-value">wozai (我宅)</span></div><div class="article-metaline-right"><span class="article-meta-tag">看板</span><span class="article-meta-value">Gossiping</span></div><div class="article-metaline"><span class="article-meta-tag">標題</span><span class="article-meta-value">Re: [新聞] 中越發佈聯合公報　越南承諾不與台灣發展</span></div><div class="article-metaline"><span class="article-meta-tag">時間</span><span class="article-meta-value">Thu May 18 17:14:52 2017</span></div>
<span class="f1 hl">有關中共與越南的聲明，媒體話只說了一半</span>，
本文稍微說明一下中國、東南亞、台灣之間的關係發展。
<span class="f3 hl">文章略長，但揭露了許多媒體不敢說的事情。</span>

中國、越南、台灣三邊關係，其實已經發展多年，

而<span class="f1 hl">中國在東南亞各國打壓台灣的事情，更已經持續幾十年。</span>

像是：

<span class="f4 hl">1999年2月，當時台灣是李登輝政府</span>，中共與泰國簽署了：

<span class="f1 hl">《中華人民共和國與泰王國關於二十一世紀合作計畫的聯合聲明》</span>

其中的第十二項：

<span class="f3 hl">「中方表示將充分尊重泰國的獨立、主權和領土完整。</span>
<span class="f3 hl">泰方重申承認中華人民共和國政府為中國的唯一合法政府，</span>
<span class="f3 hl">承認中國政府關於只有一個中國，臺灣是中國不可分割的一部份的立場。」</span>

而到了<span class="f4 hl">2012年4月，馬英九政府</span>期間，中共再次與泰國簽署：

<span class="f1 hl">《中華人民共和國與泰王國關於建立全面戰略合作夥伴關係的聯合聲明》</span>

其中的第四項特別提到：

<span class="f3 hl">「泰方繼續堅持一個中國政策，</span>
<span class="f3 hl">承認中華人民共和國政府是代表權中國的唯一合法政府，</span>
<span class="f3 hl">承認臺灣是中國領土不可分割的一部份，支持兩岸關係和平發展。」</span>

焦點回到越南：

<span class="f4 hl">2015年11月6日，一樣是馬政府時期</span>，

<span class="f1 hl">（注意：是馬英九最愛拿來說嘴的馬習會前夕。）</span>

中國與越南在河內所發表的<span class="f1 hl">《中越聯合聲明》</span>的內容是：

<span class="f3 hl">「越方重申堅定奉行一個中國政策，</span>
<span class="f3 hl">支持兩岸關係和平發展與中國統一大業，</span>
<span class="f3 hl">堅決反對任何形式的『台獨』分裂活動。越南不同台灣發展任何官方關係。</span>
<span class="f3 hl">中方對此感到贊賞。」</span>

而<span class="f2 hl">2016年蔡英文政府執政</span>後，開始推動新南向政策，

積極發展與東協及南亞國家的關係時，

<span class="f2 hl">2017年1月14日</span>，越共總書記阮富仲前往北京訪問，簽署了：

<span class="f1 hl">《中越聯合公報》</span>

其中第七點強調：

<span class="f3 hl">「越方重申堅定奉行一個中國政策，</span>
<span class="f3 hl">支持兩岸關係和平發展與中國統一大業，</span>
<span class="f3 hl">堅決反對任何形式的『台獨』分裂活動。越南不同台灣發展任何官方關係。</span>
<span class="f3 hl">中方對此感到贊賞」。</span>

眼尖的人可能已經發現了，<span class="f3 hl">2015年的聲明跟2017年的公報，內容幾乎一樣啊！</span>

而這次媒體拿來做文章的公報，

是<span class="f2 hl">2017年5月</span>，越南與中國簽的第二份<span class="f1 hl">《聯合公報》</span>，

其中第七點強調：

<span class="f3 hl">「越方重申堅定奉行一個中國政策，</span>
<span class="f3 hl">支持兩岸關係和平發展與中國統一大業，</span>
<span class="f3 hl">堅決反對任何形式的『台獨』分裂活動。越南不同台灣發展任何官方關係。</span>
<span class="f3 hl">中方對此感到贊賞」。</span>

<span class="f3 hl">內容也跟2015年馬政府時期的如出一徹。</span>

這些公報內容告訴了我們幾件事：

<span class="f3 hl">1.不管是國民黨還是民進黨執政，中共都無所不用其極的在打壓台灣。</span>

<span class="f3 hl">2.但在國民黨執政時，這些打壓容易被媒體忽視，反而在民進黨執政後，</span>
<span class="f3 hl">明明是同樣內容的公報，卻被解讀成：因為民進黨執政，所以中共打壓台灣。</span>

<span class="f3 hl">3.事實證明，中共的打壓不分藍綠，</span>
<span class="f3 hl">只是以前國民黨政府還沒被打壓就已經把膝蓋跪下去，</span>
<span class="f3 hl">看起來比較矮，所以視覺上比較沒被打壓，但只是自己騙自己。</span>

<span class="f3 hl">4.我想中國打壓的問題，重點不在於藍或綠執政，</span>
<span class="f3 hl">或是有沒有92共識，甚至是膝蓋有沒有跪下去。</span>
<span class="f3 hl">重點在於，台灣內部夠不夠團結，能不能不受媒體偏頗論述的影響。</span>

以上淺見，請參考。
