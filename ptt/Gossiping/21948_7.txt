是這樣子的

肥宅的定義必須建立在宅男基礎上

你如果是一位比較胖的宅男

那你極有可能就是肥宅

又因此我們可以延伸出肥宅所具備的幾項特點：

<span class="f6 hl">1.矮：</span>

<span class="f3 hl">肥宅基本上就不會有什麼身高，</span>

<span class="f3 hl">體型屬於橫向發展，</span>

<span class="f3 hl">因此矮是其中一項要素，也不是絕對</span>

<span class="f3 hl">在這邊說的是視覺上會比較矮</span>

<span class="f6 hl">2.魯：</span>

<span class="f3 hl">基本上肥宅人生並沒有太大的成就，</span>

<span class="f3 hl">極可能一輩子都是輸家，</span>

<span class="f3 hl">最終落得在家啃老打手槍的悲劇。</span>

<span class="f6 hl">3.醜：</span>

<span class="f3 hl">肥宅基本上不會有什麼顏值可看，</span>

<span class="f3 hl">醜也是天經地義，這邊我們不再贅述。</span>

<span class="f6 hl">4.臭：</span>

<span class="f3 hl">肥宅受體型影響，養成了懶惰的習慣，</span>

<span class="f3 hl">沒什麼衛生觀念，具備強烈的體臭，</span>

<span class="f3 hl">聞起來就像是餿水一樣，本身就不討喜。</span>


<span class="f6 hl">5.窮：</span>

<span class="f3 hl">肥宅已經相當的墮落了，</span>

<span class="f3 hl">成天在家裡無所事事，</span>

<span class="f3 hl">如果不計算爸媽所供養的金錢，</span>

<span class="f3 hl">肥宅可以說是沒有任何長處，</span>

<span class="f3 hl">更不用說那些本身爸媽就窮的死肥宅。</span>

<span class="f6 hl">6.油：</span>

<span class="f3 hl">肥宅體型屬於肥胖體質，</span>

<span class="f3 hl">體脂肪相當之高，</span>

<span class="f3 hl">油脂分泌也較一般人誇張，這邊也能延伸出油宅這個名詞。</span>

<span class="f6 hl">7.肥：</span>

<span class="f3 hl">肥宅體型本身就肥，全身贅肉，這個算是基本常識，不再贅述。</span>


<span class="f3 hl">再者，肥宅本身不討喜</span>

<span class="f3 hl">基本上不會有什麼人想要跟你接近</span>

<span class="f3 hl">因此肥宅又可能潛藏著邊緣人的屬性了</span>


<span class="f6 hl">結論就是：</span>

<span class="f6 hl">矮、魯、醜、窮、臭、油、肥</span>

<span class="f6 hl">七個基本元素在內</span>

<span class="f6 hl">再加上「邊緣人」的屬性，</span>

<span class="f6 hl">就有資格能稱作肥宅王了呦！</span>
