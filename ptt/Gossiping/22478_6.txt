A片在近代科技發展是有巨大貢獻的

想當年自組電腦，聲霸卡自己來，IRQ自己跳
是誰把一切整合？ Win OS
它整合了 driver 概念
一塊卡再也不能簡單的隨插即用，在 DOS 下驅動
但麻煩的安裝完後，所有硬體就在 os 的管理之下

聲音先提升，再來是影像，然後就進入多媒體時代
現在不強調多媒體了，就好像現在買車不會強調電動車窗
但現在的電腦是多媒體規格無誤

音樂 CD 也在這時發展，而一片 CD 的資料量，當初是因為貝多芬第九交響曲定義的
一定要能完整把它播放完; 也就是 74分 42秒 以上

現在一塊 SD 卡幾M幾G都不算什麼了，但其實它比我當年的硬碟還大，比光碟還大
又快又便宜，絕對屌打光碟
但歷史就是要慢慢進步，如果不曾有貴貴的 CD，很難一步登天到現在的 SD 卡

貝多芬第九交響曲推動了 CD 規格
而什麼推動下一步的進步呢？
就是 AV 片

---------------------

音樂 CD之後，當大家還在為&#39;輕輕震動就會跳針&#39;的問題困擾時
資訊電腦界早就瞄準它的儲存性質，想要把資料放上去，變成資料 CD
說到&#39;輕輕震動就會跳針&#39;，後來有不少科技鎖定在這裡
而最終的驗收環境，就是台灣的戰車級路面

&#34;只要在台灣的馬路播放不會跳針，就是夠好的避震規格&#34;

這點日本人也是拍馬難及的，台灣才能催生最好的避震規格
扯遠了，回到資料 CD
當年的軟體，一套 autocad 只要兩張軟碟片
就算是大到誇張的套裝軟體，十片二十片在安裝的，我也有經驗
就算你三十片好不好， 30* 1.2M = 36M
容量極小，無法催生 CD 片的需求
沒有量，就難以壓制價格，當年的 CD 空白片有夠貴!!

電影，電影不錯吧？其實 DVD 的規格已經在思考了
它的定義至少在&#34;一部電影&#34;，不能看完一部電影的 DVD，算什麼 DVD 嘛～
但就算以當年較差解析度的視訊規格，都還必需要兩張光碟片才能儲存
也就是看到一半要換片，非常不爽

一片資料 CD（儲存視訊的，又叫做 VCD），能放什麼電影？
太短了不好看，又無法刺激市場購買

也就是這時，AV 片出現了～～～
一片 AV，不必講劇情，只要女主角漂亮
一張床，一個導演，導演自己可以兼男演員（也有的是女演員在兼）
一個攝影師，極少的成本就可以開拍了
然後賺多少？以 C/P來算真是海放鐵達尼那種片子！

AV 片的出現，成功的補進這個產業需求空缺
終於讓 CD 片，CD 機不斷發展，不斷降價

現在 AV 片又在催生虛擬實境了
它對產業的貢獻真是數一數二

----------------------------------------

離題到 音樂 CD，當年本來是音樂錄音帶，一片 120
自 CD 出現之後，一片 300
說真的很讓人肉痛，但因為音質不容易損壞，還是值得
有很多人在音樂錄音帶時代，是會一次買兩片的
一片馬上聽，一片開都不開，就收藏
到音樂 CD 後，初期還是有很多刮傷意外，但愈來愈不容易
燒錄機的發明，更是讓我把原版片收起來，只聽燒錄片

但也因為盜拷容易，音樂產業受創極重
唱片公司倒閉，合併的一大堆
音樂就轉到網路，線上，數位

這也是今年金曲獎的重點
全面開放數位音樂作品
這是什麼意思呢？

從前的音樂作品，都要有實體販售
而 CD 開模（任何要開模的成品，都會有開模費）
聽說合理的低消是一千張
賣不到一千張，則開模費無法吸收

在從前一張專輯隨便賣就數萬，數十萬，甚至百萬張的年代
當然沒有問題
但盜版再加上有些人的音樂非主流，量不大
賣一千張真的有難度，若沒有金主奧援，就只好不發行實體了
這種音樂，還是可以在網路上（比如 kkbox, Apple Store）販售的

但它就是不能報名（以前的）金曲獎！

這就是規則，而今年全面開放數位音樂
也就是今年沒有發行實體的專輯，也可以報名金曲獎了

說到金曲獎，最近公佈了入圍名單
我支持的歌手也入圍了
就是黃妃 （撒花～）

黃妃前不久問我，入圍後網路上有沒有在聊她
我說:你都沒什麼誹聞，沒什麼好聊的，你上一篇新聞是&#39;買房&#39;
後來我搜尋了一下板面，說:好多人台語都好爛啊，你不如來教大家台語
於是就有了這一段錄音
這是非常生活化的，她只是隨口聊聊
等我要求她錄起來時，她就&#39;改姓張&#39;了
說她會緊張 XDDDD

唉，黃妃就是這樣，在地的親切，不習慣大場面
所以大家也別怪她低調了
